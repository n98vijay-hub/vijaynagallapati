// PHANTOM · PRIVACY PRODUCT

Phantom — Communicate Without Fear

Built for activists, journalists, and anyone who values free speech. Every message encrypted client-side. Server never sees plaintext.

🔐 AES-256-GCM Encryption

Web Crypto API. PBKDF2 key derivation with 100,000 iterations. Real cryptography — not a wrapper around someone else's library. Client-side only. Server never sees plaintext.

🌐 4-Hop Relay Network

Messages route through international relay nodes with live latency display. 8-second hop rotation. Your ISP sees encrypted traffic to Node 1 — nothing else.

🧮 Calculator Disguise

The app looks like a calculator. Enter your panic PIN → full forensic wipe. Enter your real PIN → the messenger appears. Under duress, it's just a calculator.

💥 7-Step Forensic Wipe

Server data → localStorage → sessionStorage → IndexedDB → Cache API → Service Workers → Cookies. Everything gone. No traces. No recovery.

📱 Disappearing Messages

Configurable auto-delete timers. Messages self-destruct after read. No screenshots (screen capture API detection). Forensic-resistant by design.
