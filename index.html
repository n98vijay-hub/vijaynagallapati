ENCRYPTED MESSENGER

Phantom Messenger

Military-Grade Privacy

Messages are encrypted with AES-256-GCM before leaving the device. Routed through 4-5 relay nodes so no single server sees both sender and content. If compromised, the panic calculator disguise + 7-step forensic wipe leaves no trace.

🔐
AES-256-GCM + PBKDF2

100,000 iteration key derivation. Web Crypto API — no third-party crypto libraries.

🌐
Multi-Hop Relay

4-5 node routing with randomized timing. Sender and content never visible at same node.

🧮
Panic Calculator

App disguises as a working calculator. Enter secret code → wipes all messages in 2 seconds.

📸
Encrypted Photo Sharing

Images encrypted client-side before upload. Only recipient can decrypt. No plaintext on server.
